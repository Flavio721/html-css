HTML 5 e HTML

O HTML 5 é a versão mais atualizada de HTML que temos, nela varios comandos que funcionavam no HTML 4, aqui nesta versão não são mais eficientes, por mais que funcionem, é recomendado não usa-las por serem "extintas" e causarem uma pior busca pelos mecanismos de busca do google.

Um exemplo é o bgcolor, que servia para mudar a cor de fundo, atualmente, usamos o style para personalizar várias partes de nosso site, como a cor da fonte, a própria cor do fundo e etc.

Listas

Podemos criar listas com os códigos ol e ul, o código ol é para listas com ordem por exemplo:

  1. Acordar
  2. Arrumar a cama
  3. Escovar o dente
  4. Ir para escola

Podemos mudar a forma de enumerar usando type antes de fechar o comando ol

No caso do código ul são listas sem ordem, por exemplo:

  • Xbox
  • Playstation
  • Nintendo switch

É possivel, nos dois tipos de lista, criar uma lista em um item dentro de uma lista, por exemplo:

  • Xbox
    • Forza Horizon
  • Playstation
    • God of war
    • Genshin Impact
  • Nintendo Switch
    • Mario bross
    • Zelda
